# AI Proposal Builder — Prototype

This repo contains a minimal prototype:

- `frontend/` — React + Vite frontend
- `backend/` — Express backend calling OpenAI

## Run locally

1. Backend

   - `cd backend`
   - `npm install`
   - create `.env` containing OPENAI_API_KEY
   - `node index.js`

2. Frontend
   - `cd frontend`
   - `npm install`
   - `npm run dev`
